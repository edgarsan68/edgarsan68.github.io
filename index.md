## üìú Markdown Basics

Markdown is a lightweight markup language that allows you to format plain text. Here are some basics to help you get started:

- **Headers**: Use `#` for H1, `##` for H2, and so on.
- **Bold Text**: Use `**text**` for bold.
- **Italic Text**: Use `*text*` for italics.
- **Links**: Format links as `[link text](URL)`.
- **Lists**: Use `-` for bullet points or `1.` for numbered lists.
